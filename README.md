# war3tools

KK 对战平台 · Y3 编辑器魔兽 RPG 地图的设计与脚本仓库。

参考玩法：小偷进化系列（无建造、英雄+助手、攻击偷金、波次 TD）。

## 文档结构

```
docs/
├── 更新文档.md              ← 个人决策与进度（主记录）
├── 地图设计总览.md          ← 【主文档】整体设计与规划
├── 系统/                    ← 【分文档】各系统详细设计
│   ├── 属性系统.md
│   ├── 英雄成长.md
│   ├── 技能与Buff.md
│   ├── 装备系统.md
│   ├── 助手系统.md
│   ├── 波次与敌人.md
│   ├── 经济与挑战.md
│   ├── 存档与天赋.md
│   ├── 商城系统.md
│   └── 平台与防作弊.md
└── 开发/
    ├── Phase1-任务清单.md
    └── AI自动化开发路径.md
```

**阅读顺序**：`更新文档.md` → `地图设计总览.md` → 按需进入 `系统/` 分文档。

## 技术栈

- **编辑器**：网易 Y3（KK 对战平台）
- **开发**：Cursor + Windows + Y3 开发助手 MCP
- **平台**：服务器存档、商城、云脚本防作弊
