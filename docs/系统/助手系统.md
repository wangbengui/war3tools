# 助手系统

> 上级文档：[地图设计总览](../地图设计总览.md)

助手（宠物）为纯辅助单位：跟随英雄、拾取合成、提供光环，不参与战斗。

---

## 1. 基础设定

| 项 | 规则 |
|----|------|
| 战斗 | **无攻击** |
| 生存 | **无敌**，不可被选中和击杀 |
| 移动 | 始终跟随英雄 |
| 定位 | F2 镜头跳转 |
| 皮肤 | 商城解锁，仅外观 |

---

## 2. 核心能力

| 能力 | 说明 |
|------|------|
| 自动拾取 | 范围 400，捡地面物品入助手背包 |
| 背包整理 | 每 10 秒自动整理；商城「自动整理」可加强 |
| 装备合成 | 3 个同级同部位 → 高一级，见 [装备系统](./装备系统.md) |
| 装备置换 | 在合成区或技能栏操作 |
| 光环 | 同时激活 **2 个**（面板切换） |

---

## 3. 光环列表

| ID | 名称 | 范围 | 效果 | 解锁 |
|----|------|------|------|------|
| `pet_aura_atk` | 攻击光环 | 600 | 友方攻击 +8% | 金币购买 |
| `pet_aura_spd` | 攻速光环 | 600 | 攻速 +10% | 金币购买 |
| `pet_aura_slow` | 寒霜光环 | 600 | 敌人移速 -15% | 金币购买 |
| `pet_aura_armor` | 破甲光环 | 600 | 敌人护甲 -5 | 金币购买 |

拾取与合成为默认开启；光环在商店或助手面板用金币解锁/升级。

---

## 4. 助手「技能」映射

| ID | 名称 | 类型 |
|----|------|------|
| `pet_pickup` | 自动拾取 | 默认开启 |
| `pet_sort` | 整理背包 | 默认 / 商城加强 |
| `pet_craft` | 合成 | 默认开启 |
| `pet_aura_*` | 四类光环 | 可选装 2 个 |

---

## 5. 多人协作

- 专属存放区：物品仅主人可拾取。
- 右键双击物品可传送至助手或英雄（参考小偷系列，后期实现）。
