# 助手系统

> [← 地图设计总览](../地图设计总览.md)  
> 助手为纯辅助单位：跟随刀客、拾取合成、提供光环，不参与战斗。

---

## 1. 基础设定

| 项 | 规则 |
|----|------|
| 战斗 | 无攻击 |
| 生存 | 无敌，不可被选中和击杀 |
| 移动 | 始终跟随刀客 |
| 定位 | F2 镜头跳转 |
| 皮肤 | 商城解锁，仅外观 |

---

## 2. 核心能力

| 能力 | 说明 |
|------|------|
| 自动拾取 | 范围 400，地面物品入助手背包 |
| 背包整理 | 每 10 秒自动整理 |
| 装备合成 | 3 个同级同槽 → 高一级，规则见 [装备系统 §3](./装备系统.md#3-合成规则) |
| 装备置换 | 合成区或技能栏操作 |
| 光环 | 同时激活 2 个（面板切换） |

---

## 3. 光环列表

| ID | 名称 | 范围 | 效果 | 解锁价 |
|----|------|------|------|--------|
| `pet_aura_atk` | 攻击光环 | 600 | 友方攻击 +8% | 500 金 |
| `pet_aura_spd` | 攻速光环 | 600 | 攻速 +10% | 600 金 |
| `pet_aura_slow` | 寒霜光环 | 600 | 敌人移速 -15% | 500 金 |
| `pet_aura_armor` | 破甲光环 | 600 | 敌人护甲 -5 | 800 金 |

升级：每级效果 +50%，最高 3 级；费用 = `基础价 × 当前等级`。

天赋 **T2-5** 可强化光环，见 [存档与天赋 §3](./存档与天赋.md#3-天赋树)。

---

## 4. 能力映射（实现用）

| ID | 名称 | 默认 |
|----|------|------|
| `pet_pickup` | 自动拾取 | 开启 |
| `pet_sort` | 整理背包 | 开启 |
| `pet_craft` | 合成 | 开启 |
| `pet_aura_*` | 四类光环 | 可选 2 个 |

---

## 5. 多人（后期）

- 物品仅主人可拾取
- 右键双击物品可传送至助手或刀客（v1.x 实现）
