## 2. 核心公式

```
攻击力   = 基础攻击 + 力量×1.0 + 装备攻击 + Buff攻击
生命值   = 基础生命 + 力量×20 + 装备生命
减伤率   = 护甲 / (护甲 + 100)
普攻伤害 = 攻击力 × 暴击判定 × 目标受伤加成

偷金/击  = 攻击力 × 0.10 × 偷金效率 × 目标偷金系数
           （金币怪/金币BOSS 系数=2.0，普通怪=1.0）

敏捷触发 = 敏捷>0 且 10% 概率 → 5 道能量波，每道 敏捷×10 物理伤害
智力触发 = 智力>0 且 10% 概率 → 5 道能量波，每道 智力×10 法术伤害
           （与普攻独立，各自 roll）

多重攻击 = 主目标 100% + 额外目标各 100%（偷金各自结算）

升级经验 = 100 × 1.15^(level-1)，上限 30 级
```

## 3. 属性叠加规则

| 类型 | 方式 | 示例 |
|------|------|------|
| 固定值 `+N` | 直接加 | +50 攻击 |
| 百分比 `+N%` | 基于当前基础+固定后乘算 | 攻击 +10% |
| 同 ID Buff | 可叠层（见技能与Buff文档） | 攻速最多 8 层 |
| 结算顺序 | 装备固定 → Buff% → 进化% | — |
