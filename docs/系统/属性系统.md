## 2. 溅射与多重

| 攻击类型 | 机制 | MVP |
|----------|------|-----|
| 近战（刀客） | **溅射**：主目标 100%，范围内副目标受 `splash_pct` 伤害 | 已实现 |
| 远程（后续） | **多重**：额外 N 个目标各 100% 伤害 | 未实现 |

溅射偷金：主目标全额；副目标 ×50%。

## 3. 伤害公式

```
物理伤害 = 攻击力 × 暴击 × 100 / (100 + 目标护甲)
法术伤害 = 法术基础 × 100 / (100 + 目标法抗)
```

| 伤害来源 | 类型 |
|----------|------|
| 普攻、裂金斩、神龙九斩 | 物理 |
| 震金诀 | 法术 |

怪物护甲与法抗见 [波次与敌人 §3](./波次与敌人.md#3-基础单位模板)。

---

## 4. 偷金

```
偷金/击 = 攻击力 × 0.10 × 偷金效率 × 目标偷金系数
```

| 目标 | 偷金系数 |
|------|----------|
| 妖兵 / 疾妖 / 铁甲鬼卒 / 弩妖 | 1.0 |
| 运宝妖驼 | 2.0 |
| 金库神兽 | 2.0（另 ×3 全局加成） |

经济用途见 [经济与挑战](./经济与挑战.md)。

## 6. 升级经验

```
1–50 级：100 × 1.12^(level-1)
51–100 级：上一级需求 × 1.08
单局硬上限：100 级
```

商店 **悟道** Buff：经验 +5%/层，上限 10 层。

| 场景 | 波 10 | 波 20 | 波 30 |
|------|-------|-------|-------|
| 无悟道 | ~12 级 | ~28 级 | ~40 级 |
| 悟道满层 | ~15 级 | ~35 级 | ~55 级 |

等级成长数值见 [英雄成长 §3](./英雄成长.md#3-等级)。

---

## 7. 属性叠加顺序

```
装备固定值 → 商店 Buff% → 觉醒% → 天赋 / 商城（局外）
```
