# 装备系统 · 刀客

> [← 地图设计总览](../地图设计总览.md) · [术语表 §10.4](../地图设计总览.md#104-刀客装备)  
> 刀客三槽装备、合成与商店价格。合成由 [助手系统](./助手系统.md) 执行。

---

## 1. 三槽定义

| 槽位 | 主要属性 |
|------|----------|
| 刀 | 攻击、偷金、溅射 |
| 刀鞘铠 | 生命、护甲 |
| 刀佩 | 三维、经济 |

各职业装备独立，不可跨职业合成。后续剑修 / 枪客 / 弓手 / 器灵各有一套。

## 3. 合成规则

同槽位、同品质，**3 件合 1 件**高一级：

```
柴刀×3 → 环首刀 → … → 神龙裂天刀
刀鞘铠、刀佩同理
```
