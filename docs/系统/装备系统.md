# 装备系统

> 上级文档：[地图设计总览](../地图设计总览.md)  
> 合成操作由 [助手系统](./助手系统.md) 执行

---

## 1. 规则概要

| 项 | 规则 |
|----|------|
| 品质 | 1 级 → 2 级 → 3 级 → MAX |
| 合成 | **3 个同级同部位** → 1 个高一级 |
| 槽位 | 武器 / 护甲 / 饰品（各 1，共 3） |
| 模板 | 固定属性，无随机词条（MVP） |
| 获取 | 商店直购 1 级、小怪掉落、装备挑战 |

---

## 2. 合成路径

```
武器：生锈短刃 ×3 → 窃金匕首 ×3 → 影锋 ×3 → 倒挂金钩
护甲：破布斗篷 ×3 → 夜行衣 ×3 → 影织护胸 ×3 → 无影衣
饰品：铜戒 ×3 → 敏捷指环 ×3 → 智者吊坠 ×3 → 聚宝盆
```

MAX 合成时播放全屏级特效。
