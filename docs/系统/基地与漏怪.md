# 基地与漏怪

> [← 地图设计总览](../地图设计总览.md) · [术语表 §10.1](../地图设计总览.md#101-地图与系统)  
> 神龙阁防（基地生命）：漏怪扣血、失败条件与上限来源。

---

## 1. 核心规则

| 项 | 规则 |
|----|------|
| HUD 显示名 | 神龙阁防 |
| 初始生命 (N1) | 100（N2=80，N3=60） |
| 漏怪扣血 | 妖兵 / 运宝妖驼 -1；铁甲鬼卒 -2；妖将 -5 |
| 失败 | 当前生命 ≤ 0 |
| 修复 | 局内不可修复 |

敌人类型见 [波次与敌人 §2](./波次与敌人.md#2-敌人类型)。

---

## 2. 提升神龙阁防上限

所有来源写入 `base_hp_bonus`，**加算**后作为开局上限；每局开局满血。

| 来源 | 效果 | 持久 |
|------|------|------|
| 天赋 T4-阁墙 | +15 / 级 | 永久 |
| 商城神龙护符 | +30 | 永久（付费） |
| 阁防积分兑换 | 见 [存档与天赋 §7](./存档与天赋.md#7-阁防积分) | 永久 |
| 通关成就 | 通关 N2 / N3 各 +10 | 永久 |

**无局内商店售卖的上限道具。**

---

## 3. 与波次设计的关系

早期约 70% 妖兵漏入属正常。中后期溅射与技能提高击杀率；满构建全 30 波预计扣血约 **60–85**，留有余量。

刷怪量见 [波次与敌人 §1](./波次与敌人.md#1-波次规则)。

---

## 4. HUD

- 显示：神龙阁防 ❤️ 当前 / 上限
- 漏怪多时边框泛红
- 不提供金币修复入口
