## 1. 波次规则

| 项 | 数值 |
|----|------|
| 总波次 | 30 |
| 刷怪阶段 | 前 50 秒 |
| 间歇阶段 | 后 10 秒停怪，UI 倒计时 |
| 开局 | 10 秒准备后开始第 1 波 |

### 1.1 刷怪量（单人基准）

| 波次段 | 每波总数 | 刷怪率 |
|--------|----------|--------|
| 1–3 | 30 | 0.6/秒 |
| 4–10 | 50 | 1.0/秒 |
| 11–20 | 75 | 1.5/秒 |
| 21–30 | 100 | 2.0/秒 |

多人按人数缩放数量与属性，见 [§5 属性缩放](#5-属性缩放)。

### 1.2 特殊波次

| 波次 | 事件 |
|------|------|
| 5, 10, 15, 20, 25, 30 | 金库神兽 |
| 10, 20 | 妖将 |
| 30 | 守宝龙王（或提前挑战） |

### 1.3 提前挑战守宝龙王

| 项 | 规则 |
|----|------|
| 最早波次 | ≥10 |
| 金币消耗 | 无 |
| 攻击力门槛 | ≥ `50 + 当前波次 × 15`（例：第 10 波需 ≥200） |
| BOSS 生命 | 提前挑战时 ×0.85 |

未达门槛时按钮灰显，提示「刀锋未利，不宜挑战」。验算见 [数值平衡验算 §7](./数值平衡验算.md#7-提前挑战守宝龙王)。

漏怪规则见 [基地与漏怪](./基地与漏怪.md)。

## 2. 敌人类型

| 类型 ID | 显示名 | 行为 | 出现条件 |
|---------|--------|------|----------|
| `creep_normal` | 妖兵 | 直线推进 | 全程 |
| `creep_fast` | 疾妖 | 高速低血 | 波 ≥3 |
| `creep_tank` | 铁甲鬼卒 | 高血高护甲 | 波 ≥5 |
| `creep_ranged` | 弩妖 | 远程 | 波 ≥10 |
| `creep_gold` | 运宝妖驼 | 高偷金 | 波 ≥2 |
| `elite` | 妖将 | 冲锋 / 护盾 | 波 10、20 |
| `gold_boss` | 金库神兽 | 无敌打桩 | 每 5 波 |
| `boss_final` | 守宝龙王 | 三阶段 BOSS | 通关 |

**配比公式**（在 [§1.1 刷怪量](#11-刷怪量单人基准) 总量下分配）：

```
妖兵 70% + 疾妖 15%（波≥3）+ 铁甲鬼卒 10%（波≥5）+ 弩妖 5%（波≥10）
每波 10% 妖兵替换为运宝妖驼（波≥2）
```

## 5. 属性缩放

```
生命(w) = 基础生命 × 1.12^(w-1) × 难度系数 × 人数系数
攻击(w) = 基础攻击 × 1.08^(w-1) × 难度系数 × 人数系数
```

| 难度 | 生命 | 攻击 |
|------|------|------|
| N1 | ×1.0 | ×1.0 |
| N2 | ×1.6 | ×1.3 |
| N3 | ×2.5 | ×1.6 |

| 人数 | 数量 | 生命 | 金币掉落 |
|------|------|------|----------|
| 1 | ×1.0 | ×1.0 | ×1.0 |
| 2 | ×1.5 | ×1.6 | ×1.3 |
| 3 | ×2.0 | ×2.2 | ×1.6 |
| 4 | ×2.5 | ×2.8 | ×2.0 |

## 6. 金库神兽

| 参数 | 值 |
|------|-----|
| 出现波次 | 5、10、15、20、25、30 |
| 持续 | 30 秒 |
| 生命 | 无敌 |
| 奖励 | `min(floor(伤害×0.0002×波次/5), 5000×波次/5)` |
| 偷金 | 普攻偷金生效，系数 ×3 |

| 波次 | 预估 30s 伤害 | 奖励 |
|------|---------------|------|
| 5 | ~3,000 | ~600 金 |
| 10 | ~12,000 | ~2,400 金 |
| 20 | ~50,000 | ~5,000 金（触顶） |
| 30 | ~120,000 | ~7,500 金（触顶） |

验算见 [数值平衡验算 §4.3](./数值平衡验算.md#43-金库神兽奖励公式)。

## 8. 守宝龙王

| 属性 (N1) | 数值 |
|-----------|------|
| 生命 | 500,000（提前挑战 ×0.85） |
| 攻击 | 120 × 难度系数 |
| 护甲 | 25 |
| 法抗 | 20 |
| 目标战斗时长 | 2–3 分钟（满构建） |

击杀守宝龙王 = 通关。失败可继续守波后重试。

| 阶段 | 生命% | 行为 |
|------|-------|------|
| P1 | 100–60% | 近战 + 召唤小怪 |
| P2 | 60–30% | 全图金币雨 + 攻速提升 |
| P3 | 30–0% | 狂暴攻击 +50%，每 10s 锁血 1s |
