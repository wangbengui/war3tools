---

## 1. 服务器存档槽

### 数值型（需防刷）

| 槽位 | 类型 | 读写 | 说明 |
|------|------|------|------|
| `talent_points` | int | 云脚本写 | 天赋点余额 |
| `total_wins` | int | 云脚本写 | 总通关次数 |
| `max_difficulty` | int | 云脚本写 | 最高通关难度 ≤3 |
| `n1_clears` / `n2_clears` / `n3_clears` | int | 云脚本写 | 各难度通关次数 |
| `total_play_time` | int | 云脚本写 | 累计游玩秒数 |

### 字符串 / JSON

| 槽位 | 说明 |
|------|------|
| `talent_tree` | 已激活天赋节点 ID 列表 |
| `achievements` | 成就完成状态 |
| `pet_skin` / `hero_skin` | 当前皮肤 ID |

---

## 2. 天赋点获取（云脚本结算）

| 事件 | 天赋点 |
|------|--------|
| 首次通关 N1 | +5 |
| 重复通关 N1 | +1 |
| 首次通关 N2 | +10 |
| 重复通关 N2 | +2 |
| 首次通关 N3 | +20 |
| 重复通关 N3 | +3 |
| 单局 ≥20 分钟未通关 | +1 |

防刷：同一难度每日天赋点上限 30。

---

## 3. 天赋树（MVP 12 节点）

各分支独立，无互斥；满点约需 36 点。

**经济系**
- T1-1：初始金币 +200（1 点）
- T1-2：偷金效率 +5%（2 点）
- T1-3：金币 BOSS 收益 +10%（3 点）

**战斗系**
- T2-1：初始攻击 +10（1 点）
- T2-2：攻击成长 +5%（2 点）
- T2-3：进化全属性 +8%（3 点）

**辅助系**
- T3-1：助手拾取范围 +100（1 点）
- T3-2：助手光环效果 +10%（2 点）
- T3-3：合成节省 1 材料（3 点）

**生存系**
- T4-1：初始生命 +200（1 点）
- T4-2：闪烁冷却 -1s（2 点）
- T4-3：挑战 CD -15%（3 点）

天赋只影响**开局或全局系数**，与局内进化数值分开。见 [英雄成长](./英雄成长.md)。

## 4. 开局读取流程

1. 读取存档槽
2. 解析 `talent_tree`，应用天赋加成
3. 读取平台地图等级（只读）
4. 合并 [商城系统](./商城系统.md) 道具加成
