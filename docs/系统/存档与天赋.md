## 1. 存档槽位

### 1.1 数值型

| 槽位 | 说明 |
|------|------|
| `talent_points` | 天赋点余额 |
| `fortress_points` | 阁防积分 |
| `base_hp_bonus` | 神龙阁防上限加成（合计） |
| `total_wins` / `max_difficulty` | 通关统计 |
| `n1_clears` / `n2_clears` / `n3_clears` | 分难度通关次数 |
| `total_play_time` | 累计游玩秒数 |

### 1.2 JSON / 字符串

| 槽位 | 说明 |
|------|------|
| `talent_tree` | 已激活天赋 ID 列表 |
| `achievements` | 成就 |
| `hero_skin` / `pet_skin` | 皮肤 |
| `selected_class` | 职业（MVP = `daoke`） |

云脚本读写规则见 [平台与防作弊](./平台与防作弊.md)。

## 2. 天赋点获取

| 事件 | 点数 |
|------|------|
| 首通 N1 / 重复 N1 | +5 / +1 |
| 首通 N2 / 重复 N2 | +10 / +2 |
| 首通 N3 / 重复 N3 | +20 / +3 |
| 单局 ≥20 分钟未通关 | +1 |

每日同难度天赋点上限 30（防刷）。

## 3. 天赋树

共 12 节点，大厅用天赋点激活，跨局永久生效。

### 经济系

| ID | 效果 | 消耗 |
|----|------|------|
| T1-1 | 初始金币 +200 | 1 点 |
| T1-2 | 偷金效率 +5% | 2 点 |
| T1-3 | 金库神兽收益 +10% | 3 点 |

### 战斗系

| ID | 效果 | 消耗 |
|----|------|------|
| T2-1 | 初始攻击 +10 | 1 点 |
| T2-2 | 攻击成长 +5% | 2 点 |
| T2-3 | 觉醒全属性 +8% | 3 点 |

### 辅助系

| ID | 效果 | 消耗 |
|----|------|------|
| T2-4 | 助手拾取范围 +100 | 1 点 |
| T2-5 | 助手光环 +10% | 2 点 |
| T2-6 | **金币收益 +10%** | 3 点 |

> T2-6 作用于：偷金、怪死亡金币、金库神兽结算、挑战金币奖励。

### 生存系

| ID | 效果 | 消耗 |
|----|------|------|
| T4-1 | 初始生命 +200 | 1 点 |
| T4-2 | 缩地冷却 -1s | 2 点 |
| T4-阁墙 | 神龙阁防上限 +15 | 2 点 |
| T4-3 | 挑战 CD -15% | 3 点 |

## 4. 开局应用流程

1. 读取存档 → 解析 `talent_tree`、`base_hp_bonus`
2. 应用天赋（初始金、偷金、阁防上限等）
3. 读取商城皮肤与加速道具
4. 生成刀客与助手

天赋不改 100 级经验曲线；局内 **悟道** Buff 可提高经验获取。

---

## 5. 天赋与刀客的关系

| 天赋类型 | 作用 |
|----------|------|
| 初始攻击 / 生命 / 金 | 开局面板 +N |
| T2-6 金币收益 | 所有金币来源 ×1.10 |
| 偷金 / 神兽收益 | 乘在偷金、金库神兽公式上 |
| 觉醒全属性 +8% | 觉醒百分比再 ×1.08 |
| 缩地冷却 | `blink_cd` -1s |
| 阁墙 | 神龙阁防上限 +15 |
| 助手拾取 / 光环 | 助手参数 |

不加技能等级，不直接赠送 Q / W / E。

---

## 6. 商城关系

商城 **修行札记**：天赋点获取 +20%，7 天。不直接售卖天赋点或阁防积分。商品见 [商城系统](./商城系统.md)。

---

## 7. 阁防积分

大厅「阁防」界面兑换，永久提升神龙阁防上限。与天赋阁墙、商城护符**加算**。

### 7.1 获取

| 事件 | 积分 |
|------|------|
| 通关 N1 / N2 / N3 | +30 / +50 / +80 |
| 单局游玩 ≥20 分钟 | +5 |
| 击杀守宝龙王 | +20 |

### 7.2 兑换

| 消耗积分 | 效果 | 限制 |
|----------|------|------|
| 50 | 神龙阁防上限 +5 | 可重复 |
| 200 | 神龙阁防上限 +15 | 每账号 3 次 |
| 500 | 神龙阁防上限 +30 | 每账号 1 次 |

兑换写入 `base_hp_bonus`。HUD 规则见 [基地与漏怪](./基地与漏怪.md)。
