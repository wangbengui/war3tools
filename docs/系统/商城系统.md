# 商城系统

> [← 地图设计总览](../地图设计总览.md)  
> 首发商品：皮肤 + 成长加速。不做伤害类付费。阁防积分见 [存档与天赋 §7](./存档与天赋.md#7-阁防积分)。

---

## 1. 首发商品

### 1.1 外观

| 商品 | 效果 |
|------|------|
| 刀客皮肤·江湖客 / 神龙袍 | 刀客模型换皮 |
| 助手皮肤·金鼠 / 貔貅 | 助手外观 |
| 普攻特效·金龙刀光 | 普攻视觉 |

### 1.2 成长加速

| 商品 | 效果 |
|------|------|
| 神龙护符 | 神龙阁防上限 +30（永久） |
| 修行札记 | 天赋点获取 +20%，7 天 |
| 开悟香 | 局内悟道 Buff 起始层数 +2（局数型） |

### 1.3 不做首发

- 伤害加成道具
- 十连觉醒类商品
- 阁防积分直售（由对局积累兑换）

---

## 2. 局内读取顺序

1. 皮肤 → 换模 / 特效
2. 神龙护符 → 累加 `base_hp_bonus`
3. 加速类 → 天赋点或经验系数

---

## 3. 上架配置

1. [KK 开发者平台](https://create.kkdzpt.com/map) 预配 SKU
2. 上线前 24 小时完成商业化配置

平台接入见 [平台与防作弊](./平台与防作弊.md)。
