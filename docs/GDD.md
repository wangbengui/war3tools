## 1. 产品定位

**一句话**：攻击偷金 + 波次防守 + 三阶进化 + 宠物辅助的动作 TD，通过存档与商城实现跨局成长。

| 项目 | 决策 |
|------|------|
| 编辑器 | Y3 |
| 人数 | 1–4 人（各人数均可通关） |
| 单局时长 | 约 30 分钟，支持提前挑战 BOSS 结束 |
| 首发难度 | N1 / N2 / N3 |
| 进化 | 局内 3 阶；难度与无尽模式后续扩展 |
| 宠物 | 纯辅助，无攻击，无敌 |

---

## 2. 核心循环

```
选难度 → 初始化英雄/宠物/UI → 10秒倒计时
    → 波次刷怪（英雄自动攻击 + 偷金）
    → 获得金币/杀敌/经验/掉落
    → 商店买 Buff / 技能 / 装备
    → 宠物捡装 + 合成 + 光环辅助
    → 每 5 波金币 BOSS → 每 N 波挑战副本
    → 三阶进化（外观 + 数值跃升）
    → 第 30 波 / 提前 BOSS → 结算 → 存档写入
```

**核心爽点**：攻速偷金、进化视觉冲击、装备合成 Max、存档/商城带来的局外成长碾压感。

---

## 3. 开局流程

### 3.1 进入游戏（0–30 秒）

1. 玩家选择难度：**N1 / N2 / N3**
2. 系统初始化：
   - 生成英雄于出生点
   - 生成宠物跟随英雄
   - 加载 NPC 商店（Buff / 技能 / 装备）
   - 加载 UI（资源栏、波次、倒计时、属性入口）
   - 读取服务器存档 + 商城道具加成
   - 应用天赋存档加成
3. 全图倒计时 **10 秒**，提示「怪物即将来袭」
4. 第 1 波开始

### 3.2 前 5 分钟节奏（设计目标）

| 时间 | 玩家行为 | 系统事件 |
|------|----------|----------|
| 0–1 min | 熟悉操作、买首个 Buff | 第 1–2 波，怪弱 |
| 1–3 min | 攒金币、买技能、宠物捡装 | 第 3–4 波 |
| 3–5 min | 首次合成装备、考虑 1 阶进化 | 第 5 波 + **金币 BOSS** |

## 4. 英雄系统

### 4.1 操作

| 按键 | 功能 |
|------|------|
| **TAB** | 打开属性面板（力/敏/智、攻击、攻速、偷金效率等） |
| **F1** | 镜头定位英雄 |
| **F2** | 镜头定位宠物 |
| **右键** | 闪烁（默认全图，可后续用存档/道具限制范围） |
| **A** | 攻击（可 A 地板移动攻击） |
| **进化键** | 待 UI 绑定（建议 **F** 或面板按钮） |

### 4.2 战斗

- **自动攻击**：默认开启，攻击范围内自动普攻
- **偷金机制**：普攻命中敌人时，按 `偷金系数 × 攻击力` 获得额外金币（参考小偷系列）
- **属性成长**：
  - 击杀获得经验 → 升级加基础属性
  - 商店 Buff 消耗金币/杀敌数
  - 装备提供属性

### 4.3 进化（局内 3 阶）

| 阶数 | 解锁条件（框架） | 数值提升 | 视觉变化 |
|------|------------------|----------|----------|
| **0→1** | 金币 + 等级 | 攻击/生命 +X%，偷金 +Y% | 模型光效一级 |
| **1→2** | 金币 + 杀敌数 | 全属性 +X%，技能强化 | 模型/特效二级 |
| **2→3** | 金币 + 装备条件 | 全属性 +X%，解锁终极被动 | 模型大幅变化 + 全屏特效 |

> 每阶需要「比较大的变化」——优先用 Y3 内置模型换皮 + 粒子特效 + 缩放，无需外部美术。

**待你确认**：进化消耗具体数值（可先按 N1 调一版，N2/N3 乘系数）。

## 5. 宠物系统

| 能力 | 说明 |
|------|------|
| 自动拾取 | 范围内地面物品自动捡入宠物背包 |
| 光环（可选装/升级） | 攻击、攻速、减速、减甲（同时激活 1–2 个，避免过强） |
| 装备置换/合成 | 在合成区或宠物技能栏操作，3 同级 → 高一级 |
| 皮肤 | 商城/存档解锁，仅外观 |
| 战斗 | **无攻击** |
| 生存 | **无敌**，不可被选中和击杀 |

宠物始终跟随英雄，F2 快速定位。

## 6. 经济与资源

### 6.1 资源类型

| 资源 | 来源 | 用途 |
|------|------|------|
| **金币** | 偷金、怪死亡掉落、金币 BOSS、挑战 | Buff、技能、装备、进化 |
| **木材** | 挑战、特殊怪掉落 | 部分技能/装备/挑战门票 |
| **杀敌数** | 击杀计数（不掉落，累计） | 商店 Buff、进化条件 |
| **经验** | 击杀 | 英雄升级 |

### 6.2 金币 BOSS（每 5 波）

- 第 5 / 10 / 15 / 20 / 25 / 30 波触发
- **不可击杀**，限时内承受伤害越高 → 金币奖励越多
- UI 显示当前累计伤害 / 预计金币

### 6.3 挑战系统

| 挑战类型 | 产出 | 备注 |
|----------|------|------|
| 金币挑战 | 大量金币 | 有 CD 或门票 |
| 木材挑战 | 木材 | |
| 杀敌挑战 | 杀敌数 | |
| 装备挑战 | 装备掉落 | 风险与收益并存 |

挑战入口：NPC 商店或独立挑战台，消耗金币/木材开启。

## 7. 装备系统

### 7.1 品质层级

```
1 级装备 ──×3──→ 2 级装备 ──×3──→ 3 级装备 ──×3──→ MAX 级装备
```

| 品质 | 获取 | 定位 |
|------|------|------|
| 1 级 | 商店购买、小怪掉落 | 前期过渡 |
| 2 级 | 合成 / 商店 | 中期 |
| 3 级 | 合成 / 装备挑战 | 后期 |
| MAX | 3 个 3 级合成 | 毕业装，带特殊被动 |

### 7.2 合成规则

- **3 个同级** → 随机或固定产出下一级（建议固定池 + 少量随机，降低挫败感）
- 合成操作：宠物技能 / 合成区域 / UI 按钮
- 专属区（可选）：参考小偷系列「黄色方砖区」，物品放入后防队友误拿

### 7.3 装备槽位（建议）

武器 / 护甲 / 饰品 × 若干 —— MVP 可先 **3 槽**，后续扩展。

> **待设计**：各等级装备名称、属性表、MAX 级特效列表（见 `数值表-装备.md`，Phase 2 再写）。

## 8. 波次系统

### 8.1 基础规则

- **总波次**：30 波
- **每波结构**：
  - 前 **50 秒**：刷怪，**2 怪/秒** → 每波约 100 只（按人数缩放）
  - 后 **10 秒**：停怪，UI 显示「下波倒计时」
- **最终 BOSS**：第 30 波结束后，或玩家主动「提前挑战 BOSS」

### 8.2 人数缩放（建议）

| 人数 | 怪物血量 | 怪物数量 | 金币掉落 |
|------|----------|----------|----------|
| 1 人 | ×1.0 | ×1.0 | ×1.0 |
| 2 人 | ×1.6 | ×1.5 | ×1.3 |
| 3 人 | ×2.2 | ×2.0 | ×1.6 |
| 4 人 | ×2.8 | ×2.5 | ×2.0 |

系数需在 Y3 内实测调整。

### 8.3 特殊波次

| 波次 | 事件 |
|------|------|
| 5, 10, 15, 20, 25, 30 | 金币 BOSS |
| 10, 20, 30 | 精英怪 / 小 BOSS（待细化） |
| 30 | 最终 BOSS |

### 8.4 提前结束

- 任意时刻可发起「挑战最终 BOSS」（建议加门槛：如至少完成 15 波或支付资源）
- 击败 BOSS → 胜利结算；失败 → 可继续守波或重试 BOSS

## 9. 难度设计（N1–N3）

| 难度 | 定位 | 怪物强度 | 存档掉落 | 解锁 |
|------|------|----------|----------|------|
| **N1** | 新手教学 | ×1.0 | 基础天赋点 | 默认开放 |
| **N2** | 标准体验 | ×1.8 | 更多天赋点 | 通关 N1 或地图等级 ≥X |
| **N3** | 硬核 | ×3.0 | 稀有天赋/成就 | 通关 N2 |

后续扩展：N4+、无尽模式（在 N1–N3 稳定后再做）。

## 10. 局外成长：存档与天赋

### 10.1 天赋存档（核心局外系统）

- 通关 / 达成成就获得 **天赋点**
- 天赋树分支示例：
  - 初始金币 +N
  - 初始攻击 +N%
  - 偷金效率 +N%
  - 进化消耗 -N%
  - 宠物拾取范围 +N
- 天赋点消耗后 **永久生效**（服务器存档）

### 10.2 其他存档项

| 存档 Key（示例） | 类型 | 说明 |
|------------------|------|------|
| `talent_points` | int | 可用天赋点 |
| `talent_unlocked` | string/json | 已点天赋 |
| `total_wins` | int | 总通关次数 |
| `max_difficulty` | int | 最高通关难度 |
| `map_level_bonus` | — | 平台地图等级自带，只读 |

### 10.3 商城道具（框架）

| 类型 | 示例 | 平衡原则 |
|------|------|----------|
| 外观 | 英雄皮肤、宠物皮肤 | 纯外观 |
| 便利 | 自动整理、十连进化 | 轻度付费 |
| 成长 | 经验加成卡、开局资源包 | 控制在 10–20% 以内 |
| 特权 | 额外天赋页 | 不影响 PVP（本作为 PVE） |

详细商品 ID 与价格档见 `存档与商城规划.md`。

## 11. 防作弊策略

| 敏感逻辑 | 方案 |
|----------|------|
| 抽奖 / 随机掉落 | Y3 **服务器随机池** |
| BOSS 奖励结算 | **云脚本**校验伤害/时间后发奖 |
| 天赋点 / 通关次数 | 存档槽 **防刷配置** + 仅云脚本可写 |
| 商城道具数量 | 平台托管，客户端只读 |
| 局内金币/经验 | 客户端运算，结算时云脚本校验增量上限 |

云脚本目录规划：`cloud_script/`（Phase 3 接入）。

## 13. 技术实现要点（Y3）

1. **波次控制器**：全局计时器 + 刷怪点配置表
2. **偷金**：攻击命中事件 → `金币 += f(攻击力, 偷金%, 目标类型)`
3. **宠物 AI**：跟随 + 周期性扫描地面物品
4. **合成**：背包检测 3 同级 → 消耗 + 生成新物品
5. **人数缩放**：开局读取 `玩家数` → 写入全局系数
6. **存档**：Y3 存档槽配置 → 开局读 / 结算写
7. **云脚本**：金币 BOSS 奖励、天赋点发放、敏感掉落

官方文档：
- Y3 总览：https://163.com/y3/docs/
- 存档设置：https://163.com/y3/docs/guides/FunctionManual/SystemFunction/archive-setting
- 云脚本：https://163.com/y3/docs/guides/FunctionManual/CloudScript
