## 1. 产品定位

**一句话**：刀客守神龙阁，普攻偷金、波次御妖、三阶觉醒，助手拾装合成，跨局天赋成长。

| 项目 | 内容 |
|------|------|
| 地图名 | 神龙阁 |
| 世界观 | 中国古代神话 |
| MVP 职业 | 刀客（近战溅射）；后续剑、枪、弓、器 |
| 编辑器 | Y3 |
| 人数 | 1–4 人 |
| 单局 | 默认 30 波（约 30 分钟）；第 10 波起可提前挑战守宝龙王 |
| 局内等级 | 上限 100 级（标准局约 35–55 级） |
| 局内觉醒 | 3 阶：开刃 → 断浪 → 神龙斩 |
| 助手 | 纯辅助、无敌、拾取合成 |

---

## 2. 核心循环

```
选难度(N1) → 刀客 + 助手 + 商店 → 10 秒倒计时
  → 波次御敌（偷金 + 溅射）
  → 获得金币 / 木材 / 杀敌 / 经验
  → 购买 Buff、技能、装备
  → 助手拾取并 3 合 1
  → 每 5 波金库神兽、挑战副本
  → 三阶觉醒
  → 第 30 波或提前挑战守宝龙王 → 结算 → 天赋 / 阁防积分
```

**核心体验**：溅射清线、觉醒视觉冲击、刀客专属装备合成、天赋跨局成长。

---

## 3. 设计原则

1. **职业分流**：近战溅射 vs 远程多重（MVP 刀客为溅射）。
2. **双防御**：怪物护甲减物理、法抗减法术。
3. **成长分层**：局内（等级 + 觉醒 + 装备）与局外（天赋 + 商城）分离。
4. **漏怪有代价**：神龙阁防归零则失败；不可修复，靠局外手段扩上限。
5. **命名统一**：全文以 [§10 术语表](#10-术语表) 为准，禁止混用旧称。

---

## 4. 开局流程

1. 选择难度 N1 / N2 / N3
2. 读取存档、天赋、商城；生成刀客、助手、商店与 HUD
3. 倒计时 10 秒 → 第 1 波妖兵

**前 5 分钟建议**：购买 Buff → 购买 Q 技能 → 第 5 波金库神兽 → 考虑开刃觉醒。

## 10. 术语表

> 全项目**唯一标准名称**。写作与实现均以此为准。

### 10.1 地图与系统

| 标准名 | 含义 |
|--------|------|
| 神龙阁 | 地图名称 |
| 神龙阁防 | 基地生命值；漏怪扣此值，归零失败 |
| 刀客 | MVP 英雄职业 |
| 助手 | 跟随拾取 / 合成的辅助单位 |
| 觉醒 | 局内三阶成长（F 键）：开刃、断浪、神龙斩 |
| 天赋 | 局外永久天赋树（天赋点激活） |
| 阁防积分 | 局外货币，兑换神龙阁防上限 |

### 10.2 敌人

| 类型 ID | 显示名 |
|---------|--------|
| `creep_normal` | 妖兵 |
| `creep_fast` | 疾妖 |
| `creep_tank` | 铁甲鬼卒 |
| `creep_ranged` | 弩妖 |
| `creep_gold` | 运宝妖驼 |
| `elite` | 妖将 |
| `gold_boss` | 金库神兽 |
| `boss_final` | 守宝龙王 |

### 10.3 刀客技能

| 槽位 | 技能名 |
|------|--------|
| 右键 | 缩地 |
| Q（二选一） | 裂金斩、风驰步 |
| W（二选一） | 震金诀、铁衣诀 |
| E（二选一） | 金雨诀、狂舞斩 |
| R | 神龙九斩（断浪觉醒后赠送） |

### 10.4 刀客装备

| 槽位 | 1 级 | 2 级 | 3 级 | MAX |
|------|------|------|------|-----|
| 刀 | 柴刀 | 环首刀 | 雁翎刀 | 神龙裂天刀 |
| 刀鞘铠 | 麻布刀鞘衣 | 铁叶刀鞘铠 | 蛟鳞刀衣 | 龙鳞宝铠 |
| 刀佩 | 磨刀石佩 | 刀魂玉 | 斩邪刀符 | 神龙刀令 |

### 10.5 战斗机制

| 术语 | 含义 |
|------|------|
| 溅射 | 刀客近战副目标伤害（默认 30%） |
| 多重 | 远程职业机制；刀客为 0 |
| 偷金 | 普攻命中按攻击力折算金币 |
| 护甲 | 减物理伤害 |
| 法抗 | 减法术伤害 |

### 10.6 禁用旧称

| 使用 | 禁止 |
|------|------|
| 觉醒 | 进化 |
| 金库神兽 | 金币 BOSS |
| 守宝龙王 | 守财奴、最终 BOSS |
| 妖兵 / 疾妖 / 铁甲鬼卒 / 弩妖 / 运宝妖驼 | 溃兵 / 斥候 / 铁甲卫 / 弩手 / 运金车 |
| 神龙阁防 | 基地血 |
| 阁防积分 | 阁符 |
