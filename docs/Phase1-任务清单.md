# Phase 1 任务清单（v0.1 MVP）

> 目标：**单人可玩 5 波**，验证「自动攻击 + 偷金 + 宠物拾取 + 基础 UI」  
> 工具：Y3 编辑器  
> 预计：3–4 周（业余时间）
