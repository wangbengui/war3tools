# Phase 1 任务清单（v0.1 MVP）

> [← 地图设计总览](../地图设计总览.md)  
> 目标：单人可玩 5 波，验证自动攻击 + 偷金 + 助手拾取 + 基础 UI。

---

## 前置

- [ ] 安装 KK 客户端 + Y3 编辑器（Windows）
- [ ] 注册 KK 开发者账号
- [ ] Y3 新建塔防/生存模板工程
- [ ] 阅读 [待确认事项](../待确认事项.md) 已锁定项
- [ ] 配置环境见 [AI自动化开发路径](./AI自动化开发路径.md)

## M1：工程与场景（第 1 周）

| # | 任务 | 验收标准 |
|---|------|----------|
| 1.1 | 创建 Y3 工程 | 能本地运行 |
| 1.2 | 搭建地形 | 出生点 + 出怪口 + 商店 |
| 1.3 | 放置刀客 | 可移动、可攻击 |
| 1.4 | 放置助手 | 跟随、无敌、无攻击 |
| 1.5 | 自动攻击 | 范围内自动普攻 |
