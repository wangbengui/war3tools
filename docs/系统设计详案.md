# 系统设计详案

> 阶段：**纯规划**（尚未进入 Y3 实现）  
> 开发环境：Cursor + Windows（后续迁移）  
> 本文档定义：属性、敌人、英雄成长、技能、装备、Buff —— 实现前须整体评审锁定

---

## 0. 设计原则

1. **主属性清晰**：力量驱动攻击与偷金，敏捷/智力提供概率触发技能，避免三维平均无特色。
2. **数值可读**：TAB 面板能看懂「下一档提升多少」，商店 Buff 用百分比为主。
3. **成长分层**：局内（等级+进化+装备）与局外（天赋+商城）分开叠加，公式统一。
4. **敌人服务经济**：小怪 = 经验+掉落；金币 BOSS = 爆发经济；精英/BOSS = 门槛检验。
5. **MVP 先少后多**：首发 1 英雄、12 种小怪变体、24 件装备模板、8 个商店 Buff，后续版本扩展。

### 1.2 核心公式（锁定后实现照抄）

```
攻击力     = 基础攻击 + 力量×1.0 + 装备攻击 + Buff攻击
生命值     = 基础生命 + 力量×20 + 装备生命
护甲       = 基础护甲 + 装备护甲（减伤率 = 护甲/(护甲+100)）
普攻伤害   = 攻击力 × 暴击判定 × 目标受伤加成
偷金/击    = 攻击力 × 0.10 × 偷金效率 × 目标偷金系数
             （目标为「金币怪/金币BOSS」时系数=2.0，普通怪=1.0）

敏捷触发   = 敏捷>0 且 10% 概率 → 5 道能量波，每道 敏捷×10 物理伤害
智力触发   = 智力>0 且 10% 概率 → 5 道能量波，每道 智力×10 法术伤害
             （与普攻独立，可同帧双触发；概率暂不叠加，取较高优先级或各 roll 一次——建议各 roll 一次）

多重攻击   = 主目标 100% 伤害 + 额外目标各 100%（偷金各自结算）
升级经验   = 100 × 1.15^(level-1)  （1→2 需 100，上限 30 级）
```

### 1.3 属性叠加规则

| 类型 | 叠加方式 | 示例 |
|------|----------|------|
| 固定值 `+N` | 直接加 | +50 攻击 |
| 百分比 `+N%` | 基于**当前基础+固定**后乘算 | 攻击 +10% |
| 同 ID Buff | 同子类取最高，或明确可叠层（最多 3 层） | 攻速 Buff 最多 3 层 |
| 进化/装备 | 先装备固定，再 Buff%，最后进化% | 见 1.2 |

## 2. 英雄成长系统

### 2.1 等级成长（1–30 级）

| 等级段 | 设计意图 | 每级提升 |
|--------|----------|----------|
| 1–10 | 前期手感 | +3 力量，+1 敏捷，+1 智力，+100 生命，+5 攻击 |
| 11–20 | 中期稳定 | +5 力量，+2 敏捷，+2 智力，+150 生命，+8 攻击 |
| 21–30 | 后期膨胀 | +8 力量，+3 敏捷，+3 智力，+200 生命，+12 攻击 |

- 满级 30；约在第 25 波前后满级（随人数微调经验获取）。
- 升级时播放短特效 + 可选「+1 三维可选」改为自动（减少操作，符合无建造快节奏）。

### 2.2 进化（局内 0→1→2→3）

| 阶 | 名称（待定） | 解锁条件 | 一次性消耗 | 属性跃升 | 视觉 |
|----|--------------|----------|------------|----------|------|
| 0→1 | **窃影** | 等级≥5 | 2000 金 + 50 杀敌 | 攻击+15%，生命+20%，偷金+10%，多重+0 | 光效一级 |
| 1→2 | **盗锋** | 等级≥15 | 8000 金 + 200 杀敌 + 1 件 2 级装 | 全属性+20%，攻速+15%，敏捷/智力触发概率+5% | 模型换色+武器特效 |
| 2→3 | **盗圣** | 等级≥25 | 20000 金 + 500 杀敌 + 1 件 3 级装 | 全属性+35%，偷金+25%，多重+1，解锁终极被动 | 大幅换模+周身特效 |

**终极被动（3 阶）**：「金雨」— 每 8 次普攻触发一次范围偷金（攻击力×0.5×范围内敌人数）。

### 2.3 局外成长（天赋，简要挂钩）

天赋只影响**开局**或**全局系数**，见 `存档与商城规划.md`。不与局内进化数值混表。

| 天赋示例 | 局内体现 |
|----------|----------|
| 初始金币 +200 | `gold` 初始值 |
| 偷金效率 +5% | `steal_rate` 初始 105% |
| 进化消耗 -10% | 进化金币消耗×0.9 |

## 4. 装备系统

### 4.1 规则 recap

- 品质：**1 级 → 2 级 → 3 级 → MAX**
- 合成：**3 个同级 → 1 个高一级**（同部位或全随机池——**建议同部位合成**，降低挫败感）
- 槽位：**武器 / 护甲 / 饰品**（各 1，共 3 件）
- 掉落：小怪低概率、装备挑战、商店直接买 1 级

### 6.2 波次结构（30 波标准局）

**每波时间轴**（已确认）：
- 0–50s：刷怪 2 只/秒 → 100 只/波（按人数缩放数量）
- 50–60s：停怪，UI 倒计时下一波

**单波组成公式**：

```
普通怪 70% + 快速 15%（波次≥3） + 坦克 10%（波次≥5） + 远程 5%（波次≥10）
每波替换 10% 普通 → 运金车（波次≥2）
```
