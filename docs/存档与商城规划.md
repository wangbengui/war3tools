# 存档与商城规划

> 配合 GDD v1 · Y3 编辑器  
> 上架前须在 KK 开发者后台完成存档槽与商品配置

---

## 1. 服务器存档槽设计

### 1.1 数值型存档（需配防刷规则）

| 槽位名 | 类型 | 读写 | 防作弊 | 说明 |
|--------|------|------|--------|------|
| `talent_points` | int | 云脚本写 / 客户端读 | 仅增、单局上限 | 天赋点余额 |
| `total_wins` | int | 云脚本写 | 仅增 | 总通关次数 |
| `max_difficulty` | int | 云脚本写 | 仅增、≤3 | 最高通关难度 |
| `n1_clears` | int | 云脚本写 | 仅增 | N1 通关次数 |
| `n2_clears` | int | 云脚本写 | 仅增 | N2 通关次数 |
| `n3_clears` | int | 云脚本写 | 仅增 | N3 通关次数 |
| `total_play_time` | int | 平台/云脚本 | 仅增 | 累计游玩秒数 |

### 1.2 字符串 / 表格式存档

| 槽位名 | 类型 | 说明 |
|--------|------|------|
| `talent_tree` | string (JSON) | 已激活天赋节点 ID 列表 |
| `achievements` | string (JSON) | 成就完成状态 |
| `pet_skin` | string | 当前装备宠物皮肤 ID |
| `hero_skin` | string | 当前装备英雄皮肤 ID |

### 1.3 天赋点获取规则（云脚本结算）

| 事件 | 天赋点 |
|------|--------|
| 首次通关 N1 | +5 |
| 重复通关 N1 | +1 |
| 首次通关 N2 | +10 |
| 重复通关 N2 | +2 |
| 首次通关 N3 | +20 |
| 重复通关 N3 | +3 |
| 单局游玩 ≥20 分钟（未通关） | +1 |

> 防刷：同一难度每日天赋点获取上限 30（可后台调整）。

## 2. 天赋树（局外）

### 2.1 分支结构（MVP 12 节点）

**经济系**
- T1-1：初始金币 +200（1 点）
- T1-2：偷金效率 +5%（2 点）
- T1-3：金币 BOSS 收益 +10%（3 点）

**战斗系**
- T2-1：初始攻击 +10（1 点）
- T2-2：攻击成长 +5%（2 点）
- T2-3：进化全属性 +8%（3 点）

**辅助系**
- T3-1：宠物拾取范围 +100（1 点）
- T3-2：宠物光环效果 +10%（2 点）
- T3-3：合成成功率/节省（3 点）— 如合成少消耗 1 个材料

**生存系**
- T4-1：初始生命 +200（1 点）
- T4-2：闪烁冷却 -1s（2 点）
- T4-3：挑战 CD -15%（3 点）

每个分支独立，无互斥；总消耗约 36 点满点（需多局积累）。

### 3.2 便利类

| 商品名 | 类型 | 价格档 | 效果 |
|--------|------|--------|------|
| 自动整理 | 永久 | U 币中档 | 背包每 10 秒自动整理 |

### 3.3 成长类（谨慎）

| 商品名 | 类型 | 价格档 | 效果 |
|--------|------|--------|------|
| 开局资源包 | 局数型 | U 币低档 | 开局金币 +500（每局 1 次，最多 10 局） |
| 经验加成卡 | 时间型 | U 币中档 | 7 天内经验 +15% |
| 天赋页扩展 | 永久 | U 币高档 | 第二套天赋预设（纯便利） |

**原则**：付费加成合计不超过免费玩家终局的 **15–20%**。

## 4. 局内商城道具读取（Y3）

开局流程：
1. 读取玩家商城道具列表
2. 应用外观类 → 换模型/特效
3. 应用成长类 → 写入局内加成 Buff
4. 便利类 → 注册自动整理计时器

云脚本可读：`MsGetPlayerItem` 校验消耗型道具。

---

## 5. 防作弊配置清单

在 Y3 存档设置中逐项开启：

- [ ] 数值型槽位全部配置「仅允许增加」或云脚本写入
- [ ] 天赋点使用「请求服务器扣除」回调（若 Y3 版本支持）
- [ ] 随机掉落走服务器随机池
- [ ] 结算天赋点仅云脚本 `_roomover` 事件发放
- [ ] 开启不同步检测日志（开发期）

---

## 6. 开发者后台操作顺序

1. 注册开发者 → 创建地图作品
2. **作者之家** → 配置存档 Key 列表（与上表一致）
3. 配置防刷规则（增量上限、每日上限）
4. 上架前 24h **预配置商城商品**并提交审核
5. 地图过审后 24h 商业化自动开启

后台入口：https://create.kkdzpt.com/map
